# Getting Started
### 1. Install Python dependencies for Flask backend & run

```
cd backend       
python3 -m venv venv
. venv/bin/activate

pip install -r requirements.txt 
flask run
```

### 2. Install Node modules for React frontend & run
```
cd frontend      
npm install
npm start
```
